## Rejection Rules

Skip the directory if it requires any of the following:

- Paid backlink placement as the primary value.
- Reciprocal links.
- Hidden redirects that obscure the official Nayovi source.
- App-store-only links when the official APK source must be visible.
- Forever-free, open-source, or generic-chatbot wording that misrepresents Nayovi.
- Unsupported usage metrics, revenue claims, investor claims, or customer logos.
- Uploading APK files, review codes, screenshots, or demo media before owner approval.
