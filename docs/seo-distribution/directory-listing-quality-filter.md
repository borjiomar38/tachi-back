# Nayovi Directory Listing Quality Filter

Use this packet before any AI-tool, APK, SaaS, startup, or app-directory submission. It is an owned checklist for deciding whether a listing creates trust or only creates low-quality backlink risk.

## Use The Directory When

- It accepts official developer, maker, or company submissions.
- It can show the official website or download URL: `https://tachiyomiat.com/download`.
- It can show pricing accurately as free trial plus paid token plans.
- It allows support, privacy, terms, and responsible-use links.
- It lets Nayovi describe itself as an Android OCR and AI translation workflow, not as a chapter host, scraper, or free manga source.
- It has visible editorial review, category fit, or real user discovery value.

## Skip The Directory When

- It requires a paid link package, reciprocal dofollow backlink, review-for-link exchange, fake rating, or hidden redirect.
- It hides the official download/support links behind mirror-first APK pages.
- It forces misleading labels such as free forever, open source, no signup, or browser-only.
- It asks for private credentials, APK signing keys, identity documents, cookies, or non-public verification screenshots.
- It republishes APK files without preserving source-of-truth links, hash context, support, and takedown paths.

## Standard Listing Copy

Short description:
`Nayovi is an Android APK and hosted OCR/AI translation workflow for manga, manhwa, and manhua readers, with free trial access, redeem-code activation, monthly token plans, and official support.`

Responsible-use note:
`Nayovi does not host or distribute chapters; it supports content users own, public-domain material, official samples, or content they have permission to process.`

Categories:
`Android app`, `AI OCR`, `manga translation workflow`, `manhwa translation`, `language technology`.

Source links:
- Official APK and install context: `https://tachiyomiat.com/download`
- Brand site: `https://nayovi.com`
- AI translation page: `https://tachiyomiat.com/translate-manhwa-ai`
- Pricing: `https://tachiyomiat.com/pricing`
- Support: `https://tachiyomiat.com/support`
- OCR checklist: `https://tachiyomiat.com/guides/comic-ocr-translation-checklist`

## Required Owner Inputs

- Logo/avatar and 3-5 approved screenshots.
- Demo video URL if available.
- Current APK filename, build label, size, and SHA-256.
- Package name, signing-certificate fingerprint, and Android developer verification status once owner confirms them.
- Directory-specific account or submission reference. Store only non-secret references in docs; store secrets in `/opt/tachi-back/.env.seo-distribution-agent` or an approved secret store.
