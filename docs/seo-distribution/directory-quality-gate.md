# Nayovi Directory Quality Gate

Use this gate before any AI-tool, APK, startup, app-review, or software-directory submission.

## Required Source Fields

- Official brand domain: `https://nayovi.com`
- Official APK/download path: `https://tachiyomiat.com/download`
- Hosted OCR/AI workflow: `https://tachiyomiat.com/translate-manhwa-ai`
- OCR QA checklist: `https://tachiyomiat.com/guides/comic-ocr-translation-checklist`
- Approved-sample pilot scope: `https://tachiyomiat.com/guides/permission-safe-manga-translation-pilot`
- Support path: `https://tachiyomiat.com/support`

## Acceptable Listing Claims

- Android APK and hosted OCR/AI translation workflow for manga, manhwa, and manhua reader workflows.
- Free trial access with monthly token-plan upgrade path.
- Does not host or distribute chapters.
- Intended for owned material, public-domain works, official samples, creator-provided pages, or content the tester has permission to process.
- Reviewer or directory access requires owner-approved package facts, screenshot rights, and review-code handling.

## Reject The Opportunity If It Requires

- Paid placement, reciprocal links, dofollow link gates, or hidden redirects as the price of review.
- Mirroring the APK as the primary call to action without official source context.
- Claiming Nayovi is forever-free, open source, Play-listed, developer-verified, benchmarked, endorsed, or package-registered before the owner confirms that fact.
- Uploading APKs, screenshots, identity documents, private metrics, review codes, or customer data without owner approval.
- User reviews, votes, comments, forum posts, or social promotion that would be artificial, incentivized, or undisclosed.

## Owner Confirmation Needed

- Package name.
- APK SHA-256 and whether it may be public.
- Signing-certificate fingerprint.
- Android developer verification or package-registration status.
- Approved screenshot/demo asset references.
- Public submitter/contact identity.
- Whether the directory may cite pricing, review-code access, and third-party testing results.
