## Profile Setup Rules

- Use only official Nayovi-owned, founder-owned, or developer-owned accounts with truthful affiliation.
- Do not create accounts automatically when signup, CAPTCHA, email/phone verification, identity checks, or terms acceptance are required.
- Do not use personal-looking personas, throwaway accounts, fake employees, or engagement automation.
- Use no-link first drafts for communities where rules are unclear.
- Use profile links only when the platform allows source-of-truth links and the public copy keeps the no-chapter-hosting boundary visible.
