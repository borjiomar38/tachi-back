## Operating Rules

- Build owned content first: articles, tutorials, comparisons, internal links, schema, and metadata.
- Prepare LinkedIn, Reddit, GitHub, forum, and directory drafts as value-first assets unless an authorized account/API workflow is explicitly configured.
- Use Reddit, GitHub, forums, directories, and communities only when the post is genuinely useful, allowed by that community, and not just a link drop.
- Never create fake accounts, buy links, use PBNs, mass-submit comments, scrape private lists, bypass CAPTCHAs, or automate platform abuse.
- Prefer a no-link draft when the community may reject promotion; add a link only when it is clearly relevant and allowed.
