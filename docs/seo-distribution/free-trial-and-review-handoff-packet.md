## Public Positioning

Nayovi is an Android APK and hosted OCR/AI translation workflow for manga, manhwa, and manhua reader workflows. It has official APK download, free trial or redeem-code activation, monthly hosted OCR/translation token plans, support paths, and permission-safe usage boundaries.

Nayovi does not host or distribute chapters. It is intended for material the user owns, public-domain works, official samples, creator-provided pages, or content the tester has permission to process.

## Review Handoff Questions

Use these before an APK directory, app-testing portal, or editor receives private access:

- Will the page preserve the official APK source link instead of mirroring the APK as the primary action?
- Can the listing show pricing, support, and responsible-use boundaries near the download or review CTA?
- Can review screenshots use only approved sample-safe material?
- Can review-code or trial access be measured separately from normal paid-reader traffic?
- Will the review avoid fake rating requests, incentivized user reviews, private-code publication, and unsupported package verification claims?
- Can any public report distinguish observed reviewer findings from Nayovi-owned claims?

## Reject Conditions

Skip the opportunity if it requires:

- Paid backlink placement, reciprocal links, hidden redirects, or ranking guarantees.
- Forever-free, open-source, app-store-only, or generic-chatbot wording that does not match Nayovi.
- APK mirroring as the primary source before owner approval.
- Private review codes, credentials, unpublished samples, screenshots, analytics, or package-signing facts in public docs.
- Public claims that Nayovi is Google Play listed, package-registered, Android developer verified, benchmarked, endorsed, or publisher-approved before the owner confirms that exact fact.
