# Nayovi APK Review Packet

This packet is for Android reviewers, app directories, newsletter editors, beta testers, and partner teams that need source-of-truth details before mentioning or testing Nayovi.

Nayovi is an Android APK and hosted OCR/AI translation workflow for manga, manhwa, and manhua reader workflows. Nayovi does not host or distribute chapters. It supports OCR and translation for content users own, public-domain material, official samples, or content they have permission to process.

## Reviewer Checks

Before publishing a listing, review, directory entry, or newsletter mention:

- Use the official download page as the source-of-truth link.
- Do not mirror the APK or point readers to unofficial APK copies unless the owner explicitly approves that distribution path.
- Confirm the current package name, APK SHA-256, signing-certificate fingerprint, and Android developer verification status with Nayovi before citing those facts.
- Use only owned, public-domain, official-sample, or permission-approved pages for OCR tests.
- Keep screenshots limited to install, activation, support, account, pricing, OCR progress, or approved-sample screens.
- Include the support link and responsible-use note near any install instructions.

## Safe Test Scope

Recommended tester flow:

1. Open the official download page.
2. Confirm source links, pricing, support, privacy, terms, and responsible-use copy are visible.
3. Install only from the official source supplied by Nayovi.
4. Activate with a review or pilot code supplied by Nayovi.
5. Test OCR using approved sample material only.
6. Record whether page flow, OCR block order, language choice, loading state, translation output, and support recovery are clear.
7. Do not publish sample pages, screenshots, hashes, package metadata, or test results beyond the owner-approved scope.
