## Review Scope

Nayovi is an Android APK and hosted OCR/AI translation workflow for manga, manhwa, and manhua reader workflows. It provides official APK download, free trial access, redeem-code activation, monthly token plans, and support paths.

Nayovi does not host or distribute chapters. Any public test, screenshot, demo, review, or listing should use owned content, public-domain material, official samples, or creator-approved pages.

## Safe Test Plan

1. Start from https://tachiyomiat.com/download, not an APK mirror.
2. Download the APK from the official button or direct endpoint.
3. Compare the downloaded file against the current SHA-256 hash.
4. Record Android version, device model, install prompt behavior, and whether the hash matched.
5. Activate with an owner-approved review or redeem code only when provided.
6. Test OCR/translation only with public-domain, official-sample, owned, or explicitly approved pages.
7. Keep screenshots limited to install, activation, settings, support, demo, or approved-sample screens.
8. Link readers back to the official download page, pricing page, support page, and responsible workflow guide.

## Listing Boundaries

Acceptable listings preserve:

- Official website and download links.
- Current APK hash and build label.
- Pricing and free-trial/redeem-code context.
- Support, privacy, terms, and takedown/source policy context.
- Responsible-use copy that says Nayovi does not host or distribute chapters.

Skip listings that require:

- Paid backlink placement as the review condition.
- Reciprocal homepage links.
- Review swaps or incentivized ratings.
- Mirror-first APK pages that hide support, pricing, or policy links.
- Misleading "forever free" or "open source" labels.
