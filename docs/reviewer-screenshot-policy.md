# Nayovi Reviewer Screenshot Policy

This document is a public-safe reviewer packet for Android app reviewers, APK directories, app-testing portals, newsletters, and partner diligence.

Nayovi is an Android APK and hosted OCR/AI translation workflow for manga, manhwa, and manhua reader workflows. Nayovi does not host or distribute chapters. Reviewers should test only owned content, public-domain material, official samples, or content where the creator or rights holder has granted permission.

## Allowed Screenshots

Reviewers may request or publish screenshots that show:

- Official APK download and install context from `tachiyomiat.com`.
- App activation, redeem-code, trial, pricing, or support flows.
- OCR progress and translation states using approved samples only.
- Before/after translation examples where the original sample is public-domain, official, owned by the tester, or explicitly approved for review.
- Error, recovery, and support states that help users understand the product honestly.

## Screenshots To Avoid

Do not publish screenshots that show:

- Pirated, leaked, scanlated, or unlicensed chapter pages.
- Full readable pages from copyrighted manga, manhwa, manhua, or webtoons unless the rights holder approved public use.
- Private user files, tokens, redeem codes, emails, account IDs, payment details, API keys, server logs, or support conversations.
- Any claim that Nayovi is an official tool from a manga publisher, reader app, platform, or rights holder unless that relationship is confirmed publicly.

## Reviewer Test Scope

A useful test should cover:

- Install confidence from the official source link.
- APK hash comparison when the current hash is available.
- Package name, signing-certificate fingerprint, and Android developer verification status only when owner-confirmed.
- Trial or redeem-code activation.
- Language selection and OCR/translation progress.
- OCR completeness, reading order, merged block quality, glossary consistency, and final readability.
- Support and refund/contact paths.

## Publication Language

Safe description:

> Nayovi is an Android APK and hosted OCR/AI translation workflow for manga, manhwa, and manhua reader workflows. It does not host or distribute chapters; tests should use owned, public-domain, official-sample, or permission-approved content.

Avoid:

- Calling Nayovi a chapter source, scanlation source, manga library, or official publisher platform.
- Recommending mirror-first APK downloads when the official source is available.
- Saying Nayovi is open source, free forever, Play Store verified, or Android developer verified unless the owner has confirmed that exact status.

## Owner Confirmation Needed

The following fields must remain pending until the owner confirms them:

- Package name.
- Signing-certificate fingerprint.
- Android developer verification status.
- Approved public screenshot set.
- Reviewer code or test-account scope.
- Any permission to upload APK files to external testing or review portals.
